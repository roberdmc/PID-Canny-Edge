
Para executar o programa, siga os passos abaixo:

  1 - Abra o arquivo Run.bat

  2 - Na caixa de seleção que abrir, selecione a imagem

  3 - Escolha um tamanho para o Gaussian Kernel

  4 - Escolha uma das opções:

      4.1 - Digite 1 para mostrar o passo a passo, exibindo cada imagem
        resultante de cada processo e, ao final, o plot com todas as 
        imagens reunidas

      4.2 - Digite 0 para exibir apenas o plot final, com todas as
	imagens reunidas.
